# message-engine-unity

## RPG風メッセージウィンドウ

### 実装済み

* メッセージを流す
* 禁則処理
* 特定の文字列によるイベント
 - @@p ページ送り
 - @@t 文字送りの一時停止

### サンプルリソース

* [空想曲線](http://kopacurve.blog33.fc2.com/)
* [きろ字](http://www.ez0.net/distribution/font/kiloji/)
